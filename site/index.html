One note per day

Each date maps to one editable markdown note. Tomorrow gets a new file automatically.

Local markdown

Daylog writes plain files under your chosen folder. No account, sync, or cloud lock-in.

Dock-first

Open the editor from Command Palette Dock, write, and leave. Autosave handles the rest.

Install

  1. Install PowerToys and enable Command Palette Dock.
  2. Install Daylog Dock from the latest GitHub Release.
  3. Reload Command Palette extensions.
  4. Add Daylog Dock to the Dock.
  5. Choose a folder and write.
