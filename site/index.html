What is the Dock?

PowerToys added a strip of extension buttons that floats above your taskbar. Pin Daylog there, one click opens the editor, autosave writes yyyy-MM-dd.md on disk.

One note per day

Each date is one markdown file. Tomorrow is automatic.

Local only

Your folder. No account or cloud.

Dock-first

Write, close, come back tomorrow.

Install

  1. Install PowerToys, enable Command Palette and Dock.
  2. Get Daylog Dock from GitHub Releases.
  3. Reload extensions, pin Daylog on the Dock, pick a folder.
